## 💻 Sobre o projeto

Esse projeto tem como objetivo os testes automatizados em Java, utilizando Mockito e JUnity, tecnologias indespensaveis para qualquer programador. Foi realizado durante o curso da Alura.

---

## ⚙️ Funcionalidades

- [x] Testes;
- [x] Testes usando Mockito;
- [x] Testes usando JUnity;
